# Task 3

`task_03.py` analyzes log files, counts entries by logging level, and can optionally show all entries for a selected level.

## Requirements covered

- command-line path argument handling
- optional log-level argument
- log line parsing into structured fields
- filtering and counting log entries by level
- formatted tabular output

## Example

```bash
uv run python -m goit_pycore_hw_05.task_03 data/sample.log error
```

For the generated Python API reference, open [API Reference](/python-api).
